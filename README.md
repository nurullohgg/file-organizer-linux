Linux File Organizer

This is a simple Bash script that organizes files from the Downloads folder into different categories based on their file extensions.
All organized files are moved into a folder called linuxproject in your home directory.

Features

Creates ~/linuxproject automatically

Creates category folders like:

  Images

  Videos

  Music

  Documents

  Archives

  Programming

  Scripts

  Others

Detects many common file types

Moves each file into the correct folder

Creates a log file (organizer.log)

Easy to run and understand
